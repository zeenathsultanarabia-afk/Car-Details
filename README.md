# Car Details Management System

## Overview

This project is a simple Python application that demonstrates Object-Oriented Programming (OOP) concepts. It stores and displays basic information about a car, including its model, seating capacity, engine type, and mileage.

## Features

* Create a car object
* Store car model
* Set seating capacity
* Set engine type
* Set mileage
* Display complete car details

## Technologies Used

* Python 3
* Object-Oriented Programming (OOP)

## How to Run

1. Save the program as `car_details.py`.
2. Open a terminal or command prompt.
3. Navigate to the project directory.
4. Run the program using:

```bash
python car_details.py
```

## Sample Output

```text
the car model is BE6, it's seating capacity is 5, engine type is petrol, and finally it's mileage is 20.
```
